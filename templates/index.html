Hello, world!

This is a sample website I created on my way to becoming an autodidactic in Flask. It features two web applications, each accessible through the menu bar above.

Bootstrap 4 was extensively utilized to complete this project. Little to no Javascript was used. The backend was implemented with SQLite 3 with Flask.

About the Project

Motivation

Flask is a popular Python-based web framework used to efficiently deliver applications and services. I wanted to learn Flask to deploy neural network models built with TensorFlow or Keras in the form of scalable web applications.

I also wanted to acquaint myself with the front-end workflow to see what it is like to be a full-stack developer. Through this project, I was able to glean an elementary understanding of how the front-end and backend interact in conjunction to deliever capable service to the user.

For the full story behind my journey, visit my perosnal blog.
